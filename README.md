## Deployment
Deployed on Heroku using Eco-Dyno's and including it in ALLOWED_HOSTS.
Then connected my GitHub Repo to the Heroku app and created a Procfile to run the app through gunicorn.

## Use of AI
AI was used sparingly in the project, primarily as a means to speed up the writing of code and also for limited testing/debugging.
It's secondary use was for design decisions such as colour schemes and also to write the content of the about page.
